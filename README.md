# 🚀 Excel Master v2.0

مشروع تعليمي شامل لتعلم Excel في بيئة لينكس (Termux) باستخدام أدوات سطر الأوامر.

## 📁 هيكل المشروع
```text
excel-master/
├── excel-master.sh          # السكربت الرئيسي
├── README.md
├── LICENSE
├── .gitignore
├── docs/                      # الدروس والوثائق
│   ├── 01-basics.md
│   ├── 02-formulas.md
│   ├── 03-functions.md
│   ├── 04-pivot-tables.md
│   ├── 05-charts.md
│   ├── 06-vba-basics.md
│   └── 07-data-analysis.md
├── math-logic/                # المنطق الرياضي
│   ├── probability.sh
│   ├── statistics.sh
│   ├── linear-regression.sh
│   └── distributions.sh
├── generators/                # مولدات البيانات
├── challenges/                # التحديات
│   ├── beginner.md
│   ├── intermediate.md
│   └── advanced.md
├── examples/                  # أمثلة جاهزة
│   ├── vlookup-example.csv
│   ├── pivot-example.csv
│   └── if-example.csv
└── utils/                     # أدوات مساعدة
    ├── colors.sh
    ├── validators.sh
    └── exporters.sh
```

5. عرض وتحليل الملفات
6. مسح البيانات

## 📊 التوزيعات المستخدمة
| التوزيع | الاستخدام |
