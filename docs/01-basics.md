# 🟢 Excel Basics

Columns: A, B, C, ..., Z, AA, AB, ... (16,384 columns)
Rows: 1, 2, 3, ..., 1,048,576
Cells: A1, B2, C3, ...

## Reference Types

| Type | Formula | Usage |
|-------|--------|-----------|
| Relative | `A1` | Changes when copied |
| Absolute | `$A$1` | Always fixed |
| Mixed | `$A1` or `A$1` | Fixed column or row |

## Important Shortcuts

| Shortcut | Function |
|----------|---------|
| `Ctrl + C` | Copy |
| `Ctrl + V` | Paste |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + L` | Filter |
| `F4` | Toggle reference type |
| `Ctrl + ;` | Insert Date |
| `Ctrl + Shift + :` | Insert Time |

## Quick Tips

1. **AutoFill**: Drag the small square in the cell corner.
2. **Flash Fill**: `Ctrl + E` - Writes for you.
3. **Go To Special**: `F5` → Special → Select empty cells.
4. **Format Painter**: Double-click for repeated use.
