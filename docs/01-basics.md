# 🟢 أساسيات Excel

الأعمدة: A, B, C, ..., Z, AA, AB, ... (16,384 عمود)
الصفوف: 1, 2, 3, ..., 1,048,576
الخلايا: A1, B2, C3, ...

## أنواع المراجع

| النوع | الصيغة | الاستخدام |
|-------|--------|-----------|
| نسبية | `A1` | تتغير عند النسخ |
| مطلقة | `$A$1` | ثابتة دائماً |
| مختلطة | `$A1` أو `A$1` | ثابت عمود أو صف |

## اختصارات مهمة

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl + C` | نسخ |
| `Ctrl + V` | لصق |
| `Ctrl + Z` | تراجع |
| `Ctrl + Shift + L` | فلتر |
| `F4` | تبديل نوع المرجع |
| `Ctrl + ;` | إدخال التاريخ |
| `Ctrl + Shift + :` | إدخال الوقت |

## نصائح سريعة

1. **التعبئة التلقائية**: اسحب المربع الصغير في زاوية الخلية
2. **Flash Fill**: `Ctrl + E` - يكتب نيابة عنك
3. **Go To Special**: `F5` → Special → اختر الخلايا الفارغة
4. **Format Painter**: انقر مرتين للاستخدام المتكرر
